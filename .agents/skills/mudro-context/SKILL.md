# Skill: Контекст проекта Mudro

Используй этот skill в начале каждой сессии, чтобы быстро восстановить контекст проекта.

## Инструкция

1. Прочитай оперативную память проекта в следующем порядке:
   - `.codex/todo.md` — текущие задачи
   - `.codex/done.md` — что уже сделано
   - `.codex/top10.md` — топ-10 значимых изменений
   - `.codex/state.md` — последнее состояние сессии
   - `.codex/antigravity_memory.md` — журнал решений Antigravity

2. Прочитай ключевые документы:
   - `AGENTS.md` — правила работы агентов
   - `Mission.md` — миссия и запреты
   - `README.md` — обзор проекта

3. Проверь текущее состояние:
   ```
   git status --short
   ```

4. После чтения кратко резюмируй:
   - Сколько открытых задач в TODO (P0/P1/P2)
   - Последнее значимое изменение из DONE
   - Текущий git-статус (чистый или есть незакоммиченные изменения)

## Когда использовать
- В начале каждой новой сессии Antigravity
- После длинного перерыва
- Перед крупной задачей, чтобы не потерять контекст
