## 2. [P0 — КРИТИЧНО] vercel.json — сломаны все API вызовы

**Проблема**: В коммите `011649a` (17 апреля 2026) убран rewrite:
```json
{
  "source": "/api/:path*",
  "destination": "/api/feed"
}
```

Без этого rewrite все запросы с фронтенда к `/api/posts`, `/api/front`, `/api/auth/*`, `/api/casino/*`
на Vercel возвращают 404 — Vercel не знает куда их роутить.

**Также**: нет SPA fallback — прямые URL типа `/casino`, `/chat` при обновлении страницы дают 404.

**Исправление**: восстановить `/api/:path*` rewrite и добавить SPA fallback.
**Файл исправлен в этом цикле**: `vercel.json` (см. раздел 7).

**Как проверить**: открыть Vercel-деплой, зайти на `/` — должна загрузиться лента с данными.
**Как откатить**: `git revert` коммита с исправлением или вернуть предыдущий vercel.json.
