# Tools

`tools/` — слой одноразовых CLI и операционных утилит.

Текущий статус:
- `tools/importers` — целевой дом для tg/vk import CLI.
- `tools/backfill` — целевой дом для backfill задач.
- `tools/maintenance` — целевой дом для dedupe/merge/операционных скриптов.

Переходный период:
- часть утилит пока находится в `cmd/*`;
- перенос выполняется поэтапно, без изменения DB-контракта и без массовых разрушающих миграций.
